# FinData Pipeline

Sistema de Engenharia e Análise de Dados Financeiros.

Este projeto implementa um pipeline completo de dados financeiros, incluindo:

- ingestão de dados de mercado
- validação e limpeza
- transformação de dados
- armazenamento em banco analítico
- engenharia de features
- treinamento de modelos de machine learning
- visualização de insights

## Arquitetura

Data Sources  
↓  
Data Ingestion  
↓  
Raw Data (Data Lake)  
↓  
Data Validation  
↓  
Data Transformation  
↓  
Data Warehouse  
↓  
Feature Engineering  
↓  
Machine Learning  
↓  
Dashboard

## Stack

- Python
- Pandas
- PostgreSQL / DuckDB
- Airflow
- Docker
- Streamlit

## Objetivo

Simular uma plataforma de dados utilizada por instituições financeiras para análise de mercado e previsão de tendências.
